# スニペット登録表 ver0.5
https://okarin0083.github.io/tb-pager/

VSCodeのスニペット（辞書登録機能みたいなの）の
plaintext.jsonにコピペする為のゴチャゴチャしたやつを
一括で作るやつです。

汎用的に使えるようにしてますが、デフォでは主目的の
RPGツクールMZ用の制御文字とかタグとかを入れてます。

---

## 🚀 使い方
１、「編集モード」ボタンONでカテゴリタイトル、項目、表示説明単語、呼出文字列の編集ができます
２、OFFで編集できなくなりますが、短縮ワードのみ編集可能です
３、キー動作
４、Shift+Enter：セル内での改行
５、tab：横方向への移動
６、入力確定後のEnter：下方向への移動
７、削除ボタン：行を削除します
８、行追加ボタン：行を追加します
９、一括生成ボタン：出力JSONに
１０、一括コピーボタン：
１１、保存／読込ボタン：作業の保存、保存データの読込を行います
