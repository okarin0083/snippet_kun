# TB Pager
https://okarin0083.github.io/tb-pager/

ティラノビルダー向けの **セリフ整形・ページ割りツール**。
ブラウザだけで動作します。自分用にchatGPTで作りました。

---

## 🚀 主な機能
- 見た目行数ベースの **自動ページ割り（例：4行ごと）**
- 段落（空行）ごとに **「…」でブロック化**
- 最終ページ以外に自動で **[p]** 挿入
- 全角=1 / 半角=0.5 の幅計算で自然な折り返し
- **キャラ名補完（#入力で候補表示）**
- **置換ルール**（台詞外に置換する設定も可能）
- ティラノタグ行（`[` で始まる行）はそのまま出力

---

## ✏️ 基本の書き方

1、入力テキスト欄に"#"を打つと下の「＃候補キャラ」で登録したキャラの名前が出ます。
ティラノように＃付きで入力されます。半角ならすぐに選択。全角で打っても確定後に選択できます。
スペースキーで"＃　"みたいな空白もキャラ名として登録できます。ナレーションとかに使えます。

2、入力テキストの右にある青ガイドは「文字数設定」。
「何文字打ったら自動折り返しするか」設定できます。
ティラノびるだーは大体メッセージウィンドウの画像サイズが
そのまま入力できる文字数の幅なので○を何個か打って同じぐらいになるように
するといいかもしれません。まだまだ調整中です。

3、同じようにメッセージウィンドウに合わせて「行数設定」で1ページの改行数を
入れてください。どっちかというとこっちの機能がキモです。
n行越える時に赤ガイドが出るので、改ページする箇所が分かりやすいようにしてます。

4、【変換する】で設定通りに変換できたらいいなあって感じです。

5、カギカッコは未入力でも大丈夫です。どっちかというと入れない人の方が多いかもですが
なんで入れたかというとSSとか書く時に便利かなーと思って無理やり入れてます。

6、置換ルールは本当に好きに使ってください。
